Place DB driver files in this directory, or update the BasicQuery.Properties.txt 
file's DBDRIVERDIR to point to a new directory.

Typical files are:

MySQL: mysql-connector-java-3.0.10-stable-bin.jar
Oracle: ojdbc14.jar, ojdbc14_g.jar
Sybase: jconn2.jar
Apache Derby: derbyclient.jar
DB2: db2jcc.jar, db2jcc_license_cu.jar, db2jcc_license_cisuz.jar
